DermaGenius

Proyecto desarrollado en la materia de Inteligencia Artificial, enfocado en el uso de redes convolucionales (YOLO y VGG16) para entrenar un modelo capaz de clasificar tres categorías: tipo de piel, afecciones cutáneas y edad biológica estimada. InkHarmony

Es una aplicación de escritorio desarrollada en Java que funciona como un reproductor musical offline, ofreciendo una experiencia personalizada. Diseñada con una interfaz intuitiva, permite gestionar canciones, artistas y listas de reproducción a través de operaciones CRUD conectadas a una base de datos SQLite. Durante el desarrollo se aplicaron patrones de diseño y una arquitectura en capas, integrando tecnologías como JavaFX, JLayer, JDBC y Scene Builder.

Gestor de películas

El proyecto consistió en el diseño e implementación de un sistema distribuido de gestión de películas, con tres nodos ubicados en Quito, Guayaquil y Cuenca, cada uno ejecutando un DBMS distinto (PostgreSQL y Oracle). El sistema aplicó técnicas de fragmentación horizontal y vertical, además de replicación unidireccional y bidireccional. Para la implementación trabajamos con Radmin VPN, conectando tres computadoras físicas simulando una red distribuida real. La interacción con el sistema se realizó desde una interfaz web intuitiva y amigable que permitía visualizar los procesos de fragmentación y replicación en tiempo real.

Man-in-the-middle

Este proyecto consistió en la recreación ética de un ataque Man-in-the-Middle en un entorno controlado para comprender vulnerabilidades a nivel de red. Utilizamos máquinas virtuales (Kali Linux como equipo atacante y Windows 11 como víctima) configuradas en modo adaptador puente para simular una red real. El objetivo fue analizar técnicas de suplantación como ARP spoofing y DNS spoofing, y demostrar cómo un atacante puede redirigir tráfico hacia un servidor falso (Apache2) para capturar credenciales, registradas y verificadas mediante herramientas de diagnóstico.
